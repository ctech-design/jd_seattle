## Powerful Admin Panel & Header Variations
Having a powerful admin panel in a free template is very rare but not with this template. JD Seattle got a very powerful admin panel which allows you to have a full control over the website. There are various features likes preloader animation, back to top button, wide/boxed layout etc that are accessible and can be customized from the admin panel with one click.â€™

Multiple header variations are also included in the template to give the freedom to customize the header as the way you want. There are two modes -Horizontal Mode and Stacked Mode. Further, each of these Header Mode consist of 3 more sub header variations to provide more flexibility.
